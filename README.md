# Paula Fernández Alcázar - Sitio Web de Abogada

Proyecto de página web estática para la abogada Paula Fernández Alcázar. Contiene una landing page moderna con secciones de inicio, sobre mí, misión, currículum, servicios y contacto.

## Estructura del proyecto

- `index.html` - Página principal con el contenido y la estructura del sitio.
- `assets/css/Style.css` - Estilos personalizados para el sitio.
- `assets/imgs/` - Imágenes usadas en el sitio.
- `assets/vendors/` - Librerías de Bootstrap, jQuery, Themify Icons e Isotope.

## Características

- Diseño responsive y basado en Bootstrap 4.
- Paleta visual en rojo oscuro y dorado.
- Sección de servicios con iconos destacados.
- Formulario de contacto con mapa integrado.
- Sección de misión, visión y valores.

## Cómo usar

1. Abrir `index.html` en un navegador.
2. O desplegar el proyecto en cualquier servidor estático.
